# TCS NQT 2026 Preparation Repository 🚀

Welcome to the ultimate preparation repository for the **TCS National Qualifier Test (NQT) 2026**! This repository is designed to help students master coding concepts frequently asked in TCS exams. 

We support multiple languages to ensure everyone can learn and contribute in their preferred stack:
- 🟢 **C++** (Primary)
- 🔵 **Java**
- 🟡 **Python**

## 📂 Repository Structure

The repository is organized by topics covering all essential concepts for TCS NQT:

- `01_Array_Manipulation/`
- `02_String_Processing/`
- `03_Digit_Number_Mathematics/`
- `04_Bit_Manipulation/`
- `05_Sorting/`
- `06_Searching_Techniques/`
- `07_Matrix_Grid_Operations/`
- `08_Simulation_and_Conditionals/`
- `09_CPP_Essentials/`

Inside each topic folder, you will find:
- `theory/`: Contains Markdown files explaining the concepts, tricks, and patterns.
- `cpp/`: C++ solutions and templates.
- `java/`: Java solutions.
- `python/`: Python solutions.

## 🤝 How to Contribute

We welcome contributions from everyone! Whether you want to add a solution in Java/Python, optimize an existing C++ code, or fix a typo in the theory notes, your help is appreciated.

Please read our **[Contribution Guidelines](CONTRIBUTING.md)** for detailed instructions on how to fork the repository, structure your code, and submit a Pull Request.
